Overview

Class that provides convenience constructors to create ad objects, such as banner.

+ createBannerWithIdentifier:size:adUnitId:rootViewController:eventHandler:

This method allows the developer to create an instance of a banner ad.

+ (nullable UIView<FSBanner> *)createBannerWithIdentifier:(nonnull NSString *)identifier size:(GADAdSize)size adUnitId:(nonnull NSString *)adUnitId rootViewController:(nullable UIViewController *)rootViewController eventHandler:(nullable void ( ^ ) ( NSString *__nonnull methodName , NSDictionary<NSString*,id> *__nonnull params ))eventHandler

Parameters

identifier

String that represents the unique identifier of the instance.
size

Struct that represents the size dimensions (height and width) of the instance.
adUnitId

The ad loader’s ad unit ID.
rootViewController

The root view controller is used to present ad click actions.
eventHandler

Callback that notifies the developer to certain ad events, such as adViewDidReceiveAd or didFailToReceiveAdWithError.

Return Value

banner object which is an instance of UIView and conforms to the FSBanner protocol.
